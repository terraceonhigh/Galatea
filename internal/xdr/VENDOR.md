# Vendored: buildbarn/go-xdr

These packages are copied verbatim from
[`buildbarn/go-xdr`](https://github.com/buildbarn/go-xdr) — the XDR codec and
pre-generated protocol stubs the charge designates as "used as-is" (the NFSv4
wire types, including the macOS-only `darwin_nfs_sys_prot` mount-flags protocol).

- **Source:** the `references/go-xdr` clone (see `../../references/README.md`).
- **License:** Apache-2.0. The upstream `LICENSE` is reproduced here as `LICENSE`.
- **Modifications**, both minimal and recorded:
  1. Import paths rewritten from `github.com/buildbarn/go-xdr/` →
     `github.com/terraceonhigh/galatea/internal/xdr/`, so the vendored copy
     resolves within Galatea's module with no external dependency.
  2. Five `fmt.Errorf` calls in `nfsv4_xdr.go` changed `%d` → `%v` where the
     `discriminant` arg is a bool (bool-discriminated unions: `no_newoffset`,
     `ns_sizechanged`, `lrs_present`, `new_lock_owner`, `nt_timechanged`). The
     upstream codegen emits `%d` on a bool, which `go vet` rejects and which would
     print `%!d(bool=…)` at runtime. A genuine cosmetic bug in an error path; the
     fix keeps Galatea's `go vet ./...` / `go test ./...` clean.

  The files carry "Code generated by go-xdr. DO NOT EDIT." — that warns against
  hand-edits that a *regen* would clobber. We are not regenerating; this is a
  vendor fork, and both edits are deliberate and logged here.
- **What was dropped:** all `*_test.go` (they pull `gomock`/`testify`/the upstream
  `internal/mock`, dev dependencies we don't carry), and every package except the
  ones the NFSv4 path needs: `pkg/runtime`, `pkg/protocols/{rpcv2,nfsv4,
  darwin_nfs_sys_prot}`, and `pkg/rpcserver` (the ONC-RPC record-marking server
  loop + AUTH_SYS authenticator).
- **`golang.org/x/sync/errgroup`**, which `rpcserver` imports, is NOT taken as a
  dependency: it is replaced by a self-contained ~25-line reimplementation at
  `internal/errgroup` (same WithContext/Go/Wait semantics), keeping Galatea
  dependency-free. `rpcserver`'s import was rewritten to point at it.

Vendored at R2a (DEC-010). Codec correctness is upstream's; a smoke test in
`internal/xdr/` proves the vendored copy round-trips.
